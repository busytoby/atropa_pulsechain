# Ahoy! Magazine Issue 13 (January 1985) — Technical & Historical Review

Published in January 1985, **Issue 13 of *Ahoy!* Magazine** kicked off the publication's second year, featuring game program listings, technical breakdowns of the Commodore operating system, and columns on game animation design.

---

## 1. Program Listings Index

The following programs were featured in the "Program Listings" section of Issue 13:

| Program Title | Target System | Author | Purpose / Architecture |
| :--- | :--- | :--- | :--- |
| **Ultra Mail** | C64 | - | An address book database manager to track mail details and contacts. |
| **Music Tutor** | C64 | - | An interactive educational utility to learn musical notes and composition. |
| **Alice in Adventureland** | C64 / VIC-20 | - | A textual parser adventure game mapping fantasy locations and item puzzles. |
| **To the Top** | C64 / VIC-20 | - | A platform arcade reflex game. |
| **Tape/Disk Transfer** | C64 | - | A back-up utility system to copy program blocks from cassette tape files into disk storage. |
| **Mid-print** | C64 | - | A utility helper for customized text printer setups. |

## 2. Technical Articles & Design Columns

*   **"Bringing the Screen to Life" by Orson Scott Card**:
    *   Explores game pacing, dynamic visual updates, and micro-animations to improve the aesthetic impact on low-memory 8-bit displays.
*   **"Expose of the VIC/64 Operating System" by Dale Graham**:
    *   A deep technical dive into kernel routines, RAM mapping, and hardware control register configurations.
*   **"Rupert Report"**:
    *   Dale Rupert's series focusing on low-level sprite control, screen coordinate translations, and collision flags.
