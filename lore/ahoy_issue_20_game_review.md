# Ahoy! Issue 20: Disintegrator & Fidgits Game Review

This document performs a breakdown of two notable game listings from **Ahoy! Issue 20 (August 1985)**:

---

## 1. Disintegrator (Barbara Schulak)
A high-intensity machine-language action shooter.

### **Game Mechanics**
* **Developer background**: Created by Barbara Schulak, who later became a prominent editor and puzzle designer for *Loadstar* disk magazine.
* **Core Loop**:
  * The player navigates a particle grid.
  * Targets drop from the top/sides of the screen, threatening to destroy the perimeter defenses.
  * The player uses direction keys to pivot and release energy blasts ("disintegrators") to vaporize incoming targets.
* **Architecture**: Written in fast 6502 Machine Language, circumventing the execution lag of BASIC loops.

---

## 2. Fidgits (Walter E. Meyers)
An educational alphabet tutorial game designed to teach pattern recognition and alphabetization.

### **Game Mechanics**
* **Developer background**: Created by Walter E. Meyers, a professor of English at North Carolina State University and science fiction analyst.
* **Core Loop**:
  * A series of character blocks ("Fidgits") slide down tracks.
  * The player must sort, match, or arrange the characters in correct alphabetical ordering before the tracks overflow.
* **Architecture**: Written in hybrid Commodore BASIC, utilizing custom character memory overrides to draw the "Fidgits" sprites using custom text character grids.
