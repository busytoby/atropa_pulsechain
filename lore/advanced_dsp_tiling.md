# Advanced Multi-Dimensional DSP Arpeggiator Tiling Specification

This document details the multi-dimensional arpeggiator tiling architecture supported by the Auncient Dysnomia VM and Vision Suite. It defines the mathematics of scaling synthesis operations across Time, Pitch, Space, and Timbre dimensions in high-performance vectorized DSP engines.

---

## 1. Dimensionality of Tiling Matrix

Traditional arpeggiators operate strictly in a 1D sequence over time. The Dysnomia DSP engine supports tiling across a 4-dimensional tensor space:

$$\text{Tiling Matrix } M \in \mathbb{R}^{T \times P \times S \times C}$$

Where:
*   **$T$ (Time Axis):** The rhythmic grid partition. Patterns are tiled horizontally across beat boundaries.
*   **$P$ (Pitch/Octave Axis):** Transposition vectors. Chord notes are tiled vertically across harmonically shifted registers.
*   **$S$ (Spatial/Panning Axis):** Multi-channel pan trajectories. Signal pathways are tiled geometrically across the stereo or surround space using phase-amplitude panning vectors.
*   **$C$ (Timbral/Cutoff Axis):** Synthesis parameter sweeps. Resonance envelope cutoff coefficients and oscillator waveshapes are tiled as a parameter matrix.

## 2. Mathematical Formulations

### A. Cumulative Phase Vectorization (LOD-0 Continuous Glissando)
Rather than executing discrete note trigger loops in Python, the frequency timeline is built as a global vectorized array $F \in \mathbb{R}^{N}$ and integrated using cumulative trapezoidal integration to guarantee phase continuity at the sample level:

$$\theta[n] = 2\pi \sum_{k=0}^{n} F[k] \Delta t$$

Computing the output oscillators directly from the phase accumulator $\theta[n]$ eliminates boundary clicks, supports detuned unison voices, and runs at compiled C speed:

$$y[n] = \tanh\left( \text{saw}(\theta[n]) + 0.5 \cdot \text{tri}(\theta[n]) \right)$$

### B. Generative Pattern Grammars (Fractal Melodies)
Dynamic note arrays are generated by string-rewriting systems (L-systems) defined by production grammars. Melodic sequences are grown from an initial axiom seed through production rules:

$$V \to \{A, B, C\} \quad \text{Rules: } \{A \to AB, B \to CA, C \to BC\}$$

Mapping characters to harmonic scales (e.g. A minor Phrygian steps) produces fractal melody grids that evolve with structural self-similarity across blocks.

### C. Spectral Frequency-Domain Synthesis (IFFT Chords)
Chords and pads are synthesized in the frequency domain. Specific bins in a complex spectrum vector $S \in \mathbb{C}^{K}$ are populated with phase-randomized harmonics, and converted back to time-domain blocks using the Inverse Fast Fourier Transform:

$$x[n] = \text{IFFT}(S)[n]$$

This supports rich detuned analog chorus pads with zero phase cancellations.
