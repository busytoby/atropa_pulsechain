# Ahoy! Magazine Issue 14 (February 1985) — Technical & Historical Review

Published in February 1985, **Issue 14 of *Ahoy!* Magazine** focused on printer interfacing, telecommunications, and multicolor sprite design. It features several ready-to-enter programs.

---

## 1. Program Listings Index

The following programs were featured in the "Program Listings" section of Issue 14:

| Program Title | Target System | Author | Purpose / Architecture |
| :--- | :--- | :--- | :--- |
| **Telelink 64** | C64 | Gordon F. Wheat | A telecommunications terminal program allowing modems to communicate with early BBS systems. |
| **Fontasia** | VIC-20 & C64 | John C. Laws | A character font editor and mapping utility for creating custom graphic tiles. |
| **Flankspeed** | C64 | Gordon F. Wheat | An advanced Machine Language (ML) helper that verifies lines of hexadecimal typing. |
| **VIC Eraser** | VIC-20 | John Stihvell | A utility designed to clear graphics and screen memory blocks. |
| **Bug Repellents** | VIC-20 & C64 | Kleinert and Barron | High-performance line verification programs to catch typing errors in BASIC listings. |
| **Insurance Agent** | C64 | Glenn Lumpkins | A cataloging and estimation database system designed for insurance agents to compute premiums. |
| **Futurewar!** | C64 | B. W. Behling | A space battle tactical game involving grid-based strategic movement. |

---

## 2. Highlighted Technical Articles

*   **"Ship to Shore"**:
    *   *Ahoy!*'s telecommunications column, describing telephone modem connections, carrier signals, and protocol negotiation (XMODEM).
*   **"Multicolor Sprites Explored"**:
    *   A deep dive into VIC-II memory registers for enabling 4-color sprites, trading horizontal resolution (double-width pixels) for color richness.
