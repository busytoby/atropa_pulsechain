# Ahoy! Magazine Issue 17 (May 1985) — Technical & Historical Review

Published in May 1985, **Issue 17 of *Ahoy!* Magazine** focused on disk utilities, assembly code optimizations, and custom hardware resets.

---

## 1. Program Listings Index

The following programs were featured in the "Program Listings" section of Issue 17:

| Program Title | Target System | Author | Purpose / Architecture |
| :--- | :--- | :--- | :--- |
| **Super Duper!** | C64 | - | A high-speed whole disk backup and copier utility. |
| **Two-Column Directory** | C64 | - | A directory organizer that prints disk files in two columns to maximize screen space. |
| **DSKDU!** | C64 | - | A disk drive sector read/write debugging utility. |
| **Raid!** | C64 / VIC-20 | - | A strategic space combat defense simulation game. |
| **DOS Plus!** | C64 | - | A custom utility extending BASIC with advanced disk drive control commands. |
| **Tile Time!** | C64 | - | A tile matching/placement puzzle strategy game. |

---

## 2. Highlighted Technical Articles

*   **"Hardware Reset Switch"**:
    *   Instructions on soldering a momentary push-button switch to the user port or serial port to trigger a warm/cold reset without power cycling.
*   **"Hardware Interrupt Wedge"** (by Cal Overhulser):
    *   Intercepting hardware IRQ vectors `$0314`/`$0315` to run custom background machine language tasks alongside standard BASIC.
