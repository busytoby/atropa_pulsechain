# Ahoy! Magazine Issue 15 (March 1985) — Technical & Historical Review

Published in March 1985, **Issue 15 of *Ahoy!* Magazine** focused on DOS mechanics, multi-screen level layout strategies, and the design of the Commodore Plus/4.

---

## 1. Program Listings Index

The following programs were featured in the "Program Listings" section of Issue 15:

| Program Title | Target System | Author | Purpose / Architecture |
| :--- | :--- | :--- | :--- |
| **Old Routine** | C64 | - | A system utility that reconstructs basic pointer links, reviving a program after a `NEW` command. |
| **Programmable Functions** | C64 | - | Allows users to assign custom macro command strings to the functional keys `F1`-`F8`. |
| **Automatic Line Nos.** | C64 | - | Automatically prints the next line number prefix while typing BASIC programs to speed up coding. |
| **Home Budget** | C64 | - | A financial calculator and spreadsheet database to log household expenses. |
| **Salmon Run** | VIC-20 & C64 | - | A colorful vertical arcade game where players guide a salmon upstream, leaping rapids and dodging predators. |

## 2. Highlighted Technical Articles

*   **"Making Multiscreen Gameboards!"**:
    *   Examines algorithms for handling tile memory, scrolling offsets, and screen transitions for multi-directional adventure games.
*   **"Inside the Plus/4!"**:
    *   An architectural breakdown of Commodore's TED chip (7501 CPU family) highlighting built-in applications and graphics/color changes compared to the VIC-II.
*   **"Commodore DOS"**:
    *   Deep dive into command channel parsing, sector allocations, directory structures, and the BAM (Block Availability Map).
